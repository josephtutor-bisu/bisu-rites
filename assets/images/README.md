# Images Folder - BISU R.I.T.E.S

This folder contains all image resources used throughout the BISU R.I.T.E.S application.

## Quick Start

**To add a login page background image:**

1. Place your image file in this directory with the filename: `login-cover.jpg`
2. The image should be 1920x1080 pixels (JPG format)
3. Refresh your browser to see the changes

That's it! The image will automatically appear as the cover on the left side of the login page.

## Image Requirements Quick Checklist

- [ ] Image resolution: 1920x1080 or higher

- [ ] File size: 200-500 KB (compressed)
- [ ] Filename: Exactly as specified above
- [ ] Content: Professional, research/academic related

## Image Optimization

Before adding your image, compress it for web performance:
- Visit: https://tinyjpg.com
- Upload your 1920x1080 image
- Download the compressed version
- Save as `login-cover.jpg`

Typical compression: 10-20 MB → 200-400 KB

## Troubleshooting

**Image not showing?**
- Check filename is exactly: `login-cover.jpg`
- Verify it's a JPG file
- Clear browser cache (Ctrl+Shift+Del)
- Hard refresh the page (Ctrl+Shift+R)

**Image looks stretched?**
- Make sure aspect ratio is 16:9 (1920x1080)
- Use a photo editor to crop if needed

**Page loads slowly?**
- Compress the image further using TinyJPG
- Target file size: 300 KB or less
